# 📖 Project Overview

The Iris Flower Classification project is one of the most popular beginner-friendly machine learning projects. It uses measurements of Iris flowers to predict their species.

The dataset contains **150 flower samples** belonging to three species:

- 🌸 Setosa
- 🌺 Versicolor
- 🌼 Virginica

Each flower is described using four numerical features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The objective is to train machine learning models capable of accurately classifying flower species.
