# 📖 Project Overview

Unemployment is one of the most significant economic indicators affecting a country's growth and development.

This project analyzes India's unemployment data using Python to uncover:

- 📈 Regional unemployment trends
- 📅 Monthly unemployment patterns
- 🦠 COVID-19 impact on employment
- 📊 State-wise comparison
- 🔥 Correlation between labour participation, employment, and unemployment

The project demonstrates practical data analysis and visualization techniques commonly used by Data Analysts and Data Scientists.

---

# 🎯 Objectives

- Perform Exploratory Data Analysis (EDA)
- Analyze unemployment trends over time
- Compare unemployment across Indian states
- Study the impact of COVID-19
- Create meaningful visualizations
- Draw business insights from the dataset
